# landsat-viewer-api

![Build Status](https://dev.bazile.org/ci/status/landsat-viewer-api.svg)

> Backend services for querying and viewing recent [LANDSAT 8](https://landsat.usgs.gov) scenes.

```bash
pipenv install

./scripts/develop.sh
```
